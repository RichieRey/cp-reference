VPN Debug
Procedimiento de debug para issues de VPN · Ref: sk180488
R81.10
① Iniciar debug
vpn debug trunc ALL=5 ike debug trunc ike debug on TDERROR_ALL_ALL=5
② Replicar el problema (~5 min)
③ Detener debug
vpn debug off
④ Comprimir archivos
cd $FWDIR/log tar -czvf debugvpn.tgz vpnd.elg* legacy_ike.elg* legacy_ikev2.xmll ike.elg* iked.elg* ikev2.xmll
R81.20 y Superior
① Iniciar debug
vpn debug trunc ALL=5
② Replicar el problema (~5 min)
③ Detener debug
vpn debug off
④ Comprimir archivos
cd $FWDIR/log tar -czvf debugvpn.tgz vpnd.elg* vpnd.ikev1trace* vpnd.ikev2trace* ike*.elg* iked*.ikev1trace* iked*.ikev2trace*
